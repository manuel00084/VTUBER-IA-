🏠 Inicio

Karin VTuber es una aplicación de escritorio que te permite tener una asistente VTuber con IA para tus streams de Twitch. Integra reconocimiento de voz, síntesis de voz, chat con IA y comentarista automático de juegos.

Paso 1 — Consigue tus API Keys en Groq, Cerebras y Google Studio.
Paso 2 — Conecta Twitch desde el botón rojo en la barra lateral.
Paso 3 — Configura audio en la pestaña Audio (selecciona tus dispositivos de salida).
Paso 4 — Elige una personalidad para tu asistente desde el Panel.
Paso 5 — ¡Disfruta! Activa el comentarista y el chat de Twitch.

🎤 Push-to-Talk (PTT)

Mantén presionada la tecla CTRL + F9 (configurable) para hablar, suéltala para que la IA responda.

  • Tecla: Por defecto F9, configurable desde el campo "Tecla PTT"
  • 🎤 Hablar: Botón alternativo para grabación de 4 segundos
  • Flujo: Presiona → graba Vosk STT → suelta → envía a Cerebras → responde por TTS

💬 Chat Bot IA

La pestaña más completa. Controla el chat IA de Twitch y el Modo Asistente IA para comentar juegos automáticamente.

🤖 IA para Twitch Chat

Los viewers escriben !IA + mensaje en el chat y la IA responde con voz. Configura:

  • Comando: Por defecto !IA
  • Proveedor IA: Groq, Cerebras o Google Studio IA
  • Voz TTS: Cualquiera de las 7 voces disponibles
  • 🔊 Test: Prueba la voz seleccionada

🎮 Modo Asistente IA

Analiza la pantalla del juego y comenta automáticamente. 4 modos disponibles:

1️⃣ OCR

Solo lectura de texto en pantalla (cuadros de diálogo, menús). Bajo consumo de CPU, ideal para juegos con diálogos en español.

2️⃣ OCR + Vision + IA

Combina OCR con detección visual de objetos (enemigos, barras de vida, minimapas) y comentarios generados por IA. Usa Karin Vision Lite.

3️⃣ GROQ Vision + Google Vision

Usa modelos de IA con visión (llama-3.2-11b-vision, gemini-2.0-flash) para analizar la pantalla. Máximo detalle pero consume más tokens.

4️⃣ OCR + Karin Animadora + IA

Entretenimiento y apoyo. Lee el chat de Twitch, saluda usuarios, da frases motivacionales, poesía gamer y narración épica. Usa la IA para generar frases nuevas y las guarda en karin_animadora.txt.

Categorías de Frases (Modo Animadora)

El archivo data/karin_animadora.txt contiene frases organizadas en secciones. La IA genera frases nuevas y las guarda automáticamente:

  • Frases Motivacionales Generales — Ánimo y motivación
  • Frases Motivacionales Gamer — Frases épicas de gaming
  • Poesía Gamer — Poemas sobre videojuegos
  • Narradora — Voz de cuentacuentos épica y pausada
  • Saludos Personalizados — Saluda a usuarios con [USUARIO]

Sistema anti-repetición: las frases se organizan en pools aleatorios y no se repiten hasta agotar todas.

🧠 Sistema de IA

La aplicación usa tres proveedores de IA, cada uno especializado para diferentes tareas.

Proveedores

ProveedorModelos TextoModelos VisiónFormato API
Groq llama-3.1-8b-instant, llama-3.3-70b-versatile llama-3.2-11b-vision-preview, llama-3.2-90b-vision-preview, llava-v1.5-7b-4096-preview, meta-llama/llama-4-scout-17b-16e-instruct OpenAI-compatible
Cerebras llama3.1-8b, qwen-3-235b-a22b-instruct, gpt-oss-120b — (no soporta visión) OpenAI-compatible

Funciones Principales

ask_ai()

Envía mensajes de texto al proveedor. Soporta rate limiting (429) con backoff exponencial, reintentos automáticos, y limpieza de markdown.

ask_vision()

Envía imágenes + texto a modelos con visión. Soporta Groq y Google Studio con sus formatos API específicos.

Sistema de Memoria

Memoria persistente por usuario en data/memoria.json: historial (últimos 10 mensajes), datos importantes, estado de ánimo (-5 a +5) y emoción (0-6).

Detección de Emociones

Analiza el texto del usuario para detectar: neutral, feliz, emocionado, enojado, triste, enfermo, aburrido. Ajusta voz TTS y prefijos según la emoción.

Karin Vision Lite

Sistema de detección visual optimizado para bajo consumo de CPU/RAM. Detecta:

  • Personajes: personaje_principal, personaje_secundario, enemigo, npc
  • UI: barras de vida, minimapas, botones, menús
  • Movimiento: figuras en movimiento, shake de pantalla, scroll
  • Arte: pixelart_2d, smooth_2d, dimension_2_5d, dimension_3d
  • Colores: rojo, naranja, amarillo, verde, azul, morado, oscuro, claro

Usa frame-skipping: métodos ligeros cada frame (~40ms), métodos pesados solo cada ~1s. Sin HOG ni KNN para ahorrar RAM (~288MB vs ~450MB original).
